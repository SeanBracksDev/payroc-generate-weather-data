# Payroc Generate Weather Data Techinical Exercise

## Running the app

```bash
dotnet run --project GenerateWeatherData
```

Sample output:
```
1 day(s) of weather data generated in file: 'weather_data.wis'
```

## Testing the app
```bash
dotnet test
```

## Notes
Built, run & tested using dotnet version `8.0.302`
